# Python Data Operations Projects

A portfolio of Python projects built around loan data operations,
ETL automation, SQL reconciliation, API development, and data
quality controls — aligned with a Python Developer /
Data Operations Analyst role in Consumer & Community Banking.

> All datasets used in this portfolio are entirely synthetic.
> No real customer data, bank data, or proprietary information
> is used anywhere in this repository.

## Projects

|  #  | Project | Skills demonstrated |
|-----|---------|---------------------|
| P0  | Python & Git Foundations | pathlib, logging, pytest, Git, virtual environments |
| P1  | Loan File Validator | pandas, CSV ingestion, data validation, exception reporting |
| P2  | Reconciliation Tool | SQLite, SQLAlchemy, source-vs-target reconciliation, break reports |
| P3  | FastAPI Validation Service | FastAPI, pydantic, REST endpoints, API testing |
| P4  | Streamlit Ops Dashboard | Streamlit, SLA monitoring, exception metrics, report downloads |

| P8  | LangChain Exception Assistant | LangChain, LLM chains, exception summarization, routing |
| CAP | Loan Data Ops Platform | Full platform integration, GitHub Actions CI, end-to-end delivery |
